Tag <code>

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Utilizando a tag <code>, deixamos todos os caracteres com a mesma largura, o que ajuda a escrever códigos.

Tag <pre>

Utilizada quando queremos considerar todos os espaçamentos, tornando a IDE um bloco de notas. Tudo o que estiver dentro da tag <pre> é considerado em formato de texto.
